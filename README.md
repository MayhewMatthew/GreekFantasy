# Greek Fantasy
Greek Fantasy is a Forge mod that adds tons of creatures inspired by Greek mythology.
- Satyrs
- Centaurs
- Naiads and Dryads
- Minotaurs
- Cyclopes and Gigantes
- Unicorns
- New bosses: Geryon and Cerberus
- And more!
